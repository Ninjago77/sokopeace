# sokopeace!

A Sokoban-type puzzle game oriented around a farmer organizing his animals.

Play it here:- https://ninjago77.itch.io/sokopeace

## How to play?

[WASD/Arrows]: To move
[Space]: To Restart/Reset

## AI Use

The majority of the code was done by me, except for bug fixes, advanced scene changes, & coroutine functions.
